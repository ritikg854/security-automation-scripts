# Examples

This folder contains sample input files that can be used to test and demonstrate the scripts in this repository.

## Current Contents

- `sample_iocs.txt` - sample text input containing possible IP addresses, domains, URLs, and email addresses for IOC extraction practice

## Purpose

These examples help show how the scripts in this repository can be used in simple security analysis and investigation-support scenarios.

## Example Output

When run against the sample input file, the script identifies possible:

- IP addresses
- email addresses
- URLs
- domains

This reflects a simple analyst-support workflow for extracting useful indicators from suspicious text.
