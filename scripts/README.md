# Scripts

This folder contains small security automation scripts designed to support analyst workflows, investigation tasks, and practical security operations learning.

## Purpose

The scripts in this folder are intended to demonstrate simple, job-relevant technical work aligned with SOC, blue-team, and cybersecurity analyst responsibilities.

## Current Contents

- `ioc_extractor.py` - extracts potential indicators of compromise such as IP addresses, domains, URLs, and email addresses from a block of text

## Professional Relevance

These scripts reflect practical automation work that can support security monitoring, phishing review, incident triage, and analyst efficiency.
